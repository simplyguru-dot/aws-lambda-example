
# aws-lambda-example

An example of using AWS Lambda function with Clojure

## Installation

 - Clone repo

    $ git clone git@github.com:lowl4tency/aws-lambda-example.git

 - Build app

    $ lein uberjar

 - Load Lambda function

    $ bash -x load.sh

 - Configure Event trigger in AWS Web Console
